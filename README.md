# Usage

| Key                       | Description                   |
|---------------------------|-------------------------------|
| `j`=Up `k`=Down           | Move cursor                   |
| `\n`                      | Confirm                       |
| `q`                       | Quit                          |
| `n`,`>`=Next `p`,`<`=Prev | Change page                   |
| `a` to `h`                | Switch selection              |
| `r` `w`                   | Change ReadOnly/Write         |
| `u`                       | Change to the upper directory |
| ` `                       | Switch display mode           |
| `0-9`, Enter, BS, Esc     | Jump to question              |

# Usage

| Key                     | Description                   |
|-------------------------|-------------------------------|
| j=Up k=Down             | Move cursor                   |
| Enter                   | Confirm                       |
| q                       | Quit                          |
| n,&gt;=Next p,&lt;=Prev | Change page                   |
| a to h                  | Switch selection              |
| r w                     | Change ReadOnly/Write         |
| u                       | Change to the upper directory |
